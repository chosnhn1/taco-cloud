## 1.4 스프링 살펴보기

1. 핵심 스프링 프레임워크
  * 의존성 주입 프레임워크
  * 스프링 MVC 웹 프레임워크
  * 템플릿 기반 JDBC / 기본 데이터 퍼시스턴스
  * 리액티브 프로그래밍 지원 (스프링 WebFlux)
2. 스프링 부트
  * 의존성 관리
  * 자동 구성
  * Actuator
  * 환경속성 명세
  * 테스트
  * CLI
3. 스프링 데이터
  * (기본 퍼시스턴스는 core에 포함 but 더 많은 기능들)
  * RDB: JDBC / JPA
  * 문서형: Mongo
  * 그래프형: Neo4j
  * ...
4. 스프링 시큐리티
  * 보안 프레임워크
  * Authentication & Authorization, API Security, ...
5. 스프링 통합과 배치
  * Spring Integration
  * Spring Batch
  * 
6. 스프링 클라우드
